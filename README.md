# 📱 CIS 통합 문자 발송 매크로 (번호등록 + 문자발송)

> **Windows & macOS 멀티 플랫폼을 완벽 지원**하는 프리미엄 GUI 기반 단체/개별 선택 문자 발송 매크로 프로그램입니다.

---

## 🌟 주요 특징 (Key Features)

* **💻 완벽한 크로스 플랫폼 지원**: Windows의 `Ctrl` 단축키와 macOS의 `Command` 단축키를 자동으로 감지하여 작동하며, 로그 및 폴더 열기 기능도 각 OS에 맞게 자동으로 분기됩니다.
* **🎨 프리미엄 슬레이트/네이비 테마**: 윈도우와 맥북 모두에서 버튼 글씨가 흐려지거나 테마가 깨지는 현상 없이 선명하고 일관된 고대비 UI를 제공합니다.
* **🔒 차단방지 스마트 랜덤 딜레이**: 지정한 매크로 대기 시간을 기준으로 매 발송 시마다 미세 가변 시간(설정값 기준 ±0.5초)을 무작위로 자동 주입하여 통신사의 자동 매크로 탐지(스팸 엔진)를 우회합니다.
* **📊 결과 엑셀 보고서 저장**: 전체 발송 목록의 처리 결과(대기/성공/실패)와 완료 시각이 정제된 스타일시트 양식의 엑셀 파일(`.xlsx`)로 내보낼 수 있습니다. (성공은 초록, 실패는 빨강으로 하이라이트 표시)
* **⚠️ 발송 수량 제한 가드**: 개별 통신사 스팸 정책에 부합하도록 배치당 전송 한도 건수를 설정할 수 있으며, 한도 도달 시 자동으로 일시정지 및 경고창을 띄워 번호 차단을 원천 방지합니다.
* **🧹 전화번호 자동 정제**: 로드하는 엑셀 원본 연락처에 하이픈(`-`), 공백, 국가코드(`+82`) 등이 포함되어 있어도 시스템이 로드하는 즉시 깨끗하게 포맷팅하여 누락 없는 데이터를 로드합니다.
* **📅 관리자 전용 달력 선택**: 라이선스 관리용 만료일을 기입할 때, 타이핑 없이 간편하게 네이티브 달력 위젯 팝업을 띄워 만료일을 마우스 클릭만으로 지정할 수 있습니다.
* **🖱️ 목록 더블클릭 지능형 편의성**: 발송 대기 목록의 행을 더블클릭하면 즉시 해당 건을 삭제하며, 완료 목록의 행을 더블클릭하면 대기 목록으로 다시 복원(재전송용)할 수 있습니다.
* **💡 예시 데이터 로드**: 별도의 엑셀 파일 없이도 원클릭으로 가상의 수신처 데이터를 생성하여 빠르게 테스트해볼 수 있습니다.
* **➕ 실시간 수동 등록**: 프로그램 내부에서 연락처와 문구를 즉시 추가하여 발송 대기 목록을 구성할 수 있습니다.
* **⚡ 단건 즉시 발송**: 리스트에 등록하지 않고 기입한 하나의 번호로 즉시 마우스 좌표 기반 매크로 발송을 실행할 수 있습니다.
* **📊 실시간 엑셀 기록**: 작업이 완료될 때마다 연동된 엑셀 시트의 D열에 결과를 실시간으로 기록(성공/실패)합니다.

## 🚀 다운로드 및 실행 방법 (How to Install & Run)

프로그램을 다른 사용자와 공유하여 실행할 때는 아래 단계를 따르시면 됩니다.

### Step 1. 프로그램 다운로드

* **방법 A (추천 - 무설치 단독 실행 파일)**
  파이썬 설치 번거로움 없이 즉시 실행하고 싶으시다면 공식 배포 페이지에서 본인 OS에 맞는 실행 파일을 다운로드하세요:
  👉 **[무설치 실행 파일 다운로드 (Releases 페이지)](https://github.com/heebonpark/CIS_SMS_KTT/releases/tag/latest)**
  * **Windows**: `CIS_SMS_Macro_Windows.exe`를 다운로드하여 실행합니다.
  * **macOS**: `CIS_SMS_Macro_Mac.zip`를 다운로드하여 압축 해제 후 앱을 실행합니다. (실행 전 `시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용`에서 실행 주체에 대한 제어 권한 허용이 필요합니다.)

* **방법 B (압축 파일로 소스 코드 받기)**
  1. 우측 상단의 녹색 **[Code]** 버튼을 클릭한 뒤 **[Download ZIP]**을 클릭합니다.
  2. 다운로드된 압축 파일을 해제합니다.
  
* **방법 C (개발자형 - Git 클론)**
  ```bash
  git clone https://github.com/heebonpark/CIS_SMS_KTT.git
  cd CIS_SMS_KTT
  ```

---

### Step 2. 파이썬 및 패키지 설치
이 프로그램은 Python 3를 기반으로 작동합니다. 실행을 위해 아래의 필수 패키지들을 설치해 줍니다.

1. [Python 공식 홈페이지](https://www.python.org/downloads/)에서 파이썬을 설치합니다. (설치 시 **Add Python to PATH** 체크박스 필수 선택)
2. 터미널(macOS) 또는 명령 프롬프트(Windows)를 열고 아래 명령을 실행하여 필수 라이브러리를 설치합니다.
   ```bash
   pip install pandas pyautogui pyperclip xlwings openpyxl
   ```

---

### Step 3. 프로그램 실행
다운로드 폴더 경로로 이동하여 아래 명령으로 프로그램을 구동합니다.
* **macOS / Linux**
  ```bash
  python3 app.py
  ```
* **Windows**
  ```bash
  python app.py
  ```

---

## 📍 사용 방법 및 좌표 설정 가이드

1. **비밀번호 입력**: 초기 구동 시 보안 설정된 비밀번호(기본: `0303`)를 입력하여 접속합니다.
2. **데이터 연동**: 엑셀 연동 버튼을 누르거나, 수동 등록 또는 예시 데이터를 로드합니다.
3. **좌표 지정**: [좌표 재설정]을 누른 뒤, 카운트다운(5초) 동안 발송 화면(웹사이트 등)의 입력칸 및 버튼 위치에 마우스 커서를 올려 놓아 매크로 작동 위치를 지정합니다.
4. **매크로 실행**: 좌표 설정이 완료되면 [실행] 버튼을 클릭해 매크로를 가동합니다. (`pyautogui.FAILSAFE`가 활성화되어 있어 위급 시 마우스를 모니터 화면의 네 모서리 끝 구석으로 강하게 이동시키면 비상 중지됩니다.)
