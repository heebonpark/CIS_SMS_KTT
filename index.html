CIS 통합 문자 발송 매크로

마우스 좌표 매크로와 실시간 엑셀 결과 연동 기능을 탑재하여 대량/개별 문자 발송을 완벽히 자동화하는 데스크톱 전용 프로그램입니다.

CIS SMS Macro GUI

주요 핵심 기능 (Core Features)

💻

크로스 플랫폼 최적화

맥북의 Command 키와 윈도우의 Ctrl 단축키 동작을 운영체제별로 실시간 감지 및 분기 처리하여 완벽히 구동됩니다.

🎨

맥북 화면 렌더링 버그 해결

맥 OS 특유의 버튼 배경색 무시 오류를 극복하기 위해 커스텀 LabelButton으로 설계되어 가시성과 가독성이 뛰어납니다.

⚡

단건 즉시 발송

대기 목록 파일 없이도 화면 내부에서 직접 연락처와 발송용 개별 문구를 기입하고 원클릭으로 매크로를 즉시 가동할 수 있습니다.

📊

실시간 엑셀 백그라운드 기록

발송 처리가 한 건씩 진행될 때마다, 연동된 엑셀 파일의 D열에 성공 여부 및 시간 정보가 자동으로 즉시 마킹됩니다.

설치 및 실행 방법 (Quick Start Guide)

1

프로그램 파일 다운로드

페이지 상단의 파란색 [⚡ 프로그램 다운로드 (ZIP)] 버튼을 클릭하여 소스 코드 압축 파일을 내려받고 편리한 경로에 해제합니다.

2

필수 모듈 패키지 원클릭 설치

터미널(macOS) 또는 명령 프롬프트(Windows) 창을 실행하고, 압축 해제한 경로(예: cd Downloads/CIS_SMS_KTT)로 이동하여 라이브러리를 설치합니다.

pip install pandas pyautogui pyperclip xlwings openpyxl
3

매크로 시작

아래 명령어를 사용하여 매크로 전용 GUI를 즉시 실행합니다.

macOS 사용자:

python3 app.py

Windows 사용자:

python app.py

실행 비밀번호의 기본값은 0303입니다.
