CIS 통합 문자 발송 매크로

번호등록부터 문자 발송까지, 수작업 없는 프리미엄 자동화 매크로 프로그램

⚡ 프로그램 다운로드 (ZIP) 💾 GitHub 소스코드 보기
💻

크로스 플랫폼 지원

맥북의 Command 단축키와 윈도우의 Ctrl 단축키를 자동 분기 처리하여 양대 OS 환경 어디서나 완벽하게 구동됩니다.

🎨

프리미엄 커스텀 UI

맥북 Aqua 테마 오류를 개선한 LabelButton 설계로 고대비 가시성이 뛰어난 프리미엄 Slate/Navy 디자인을 제공합니다.

⚡

단건 즉시 발송

대기 리스트 빌드 필요 없이 화면 내에서 개별 연락처와 문구를 즉시 마우스 좌표 연동 매크로로 송출할 수 있습니다.

📊

엑셀 실시간 기록

발송이 처리될 때마다 백그라운드에서 동작하여 연동된 active 엑셀 시트 D열에 성공/실패 여부를 실시간으로 기재합니다.

설치 및 실행 가이드 (Quick Start)

1

프로그램 다운로드 및 압축 해제

우측 상단의 다운로드 버튼을 눌러 ZIP 파일을 다운로드하고, 원하는 폴더에 압축을 풉니다.

2

필수 패키지 설치

명령 프롬프트(Windows) 또는 터미널(macOS)을 열어 압축 해제한 폴더로 이동한 뒤, 아래 패키지 설치 명령어를 입력합니다.

pip install pandas pyautogui pyperclip xlwings openpyxl
3

프로그램 실행

터미널에 아래 구동 명령을 작성하여 매크로 프로그램을 시작합니다.

macOS 사용자:

python3 app.py

Windows 사용자:

python app.py

초기 보안 실행 비밀번호는 0303입니다.
