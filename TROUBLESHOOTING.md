# 🛠️ CIS 문자발송 매크로 장애 해결 가이드 (Troubleshooting)

프로그램 실행 및 매크로 구동 중 발생할 수 있는 주요 문제들과 해결 방법입니다. 크몽 판매 및 고객 지원 시 참고 자료로 활용하실 수 있습니다.

---

## 1. macOS 권한 관련 문제 (마우스 클릭/입력 안 됨)
macOS의 보안 정책으로 인해, 마우스/키보드 제어 라이브러리(PyAutoGUI)가 동작하려면 반드시 **시스템 권한**이 승인되어야 합니다.

### 증상:
- 프로그램은 실행되나, 매크로 시작 시 마우스가 움직이지 않거나 클릭이 되지 않음.
- 터미널에 `FailSafeException` 또는 `Accessibility` 관련 에러가 출력됨.

### 해결 방법:
1. macOS **시스템 설정 (System Settings)** > **개인정보 보호 및 보안 (Privacy & Security)**으로 이동합니다.
2. **손쉬운 사용 (Accessibility)** 메뉴를 클릭합니다.
3. 리스트에 있는 **터미널(Terminal)**, **VS Code**, 또는 빌드된 앱(**CIS_SMS_Macro**)의 토글을 켭니다 (잠금 해제 후 활성화 필요).
4. 만약 동일한 문제가 지속된다면, **입력 모니터링 (Input Monitoring)** 메뉴에서도 동일하게 권한을 활성화합니다.
5. 권한 변경 후 **프로그램을 완전히 종료했다가 다시 실행**합니다.

## 2. 엑셀 연동 및 결과 기록 실패 (xlwings 관련)
활성 엑셀 연동을 사용하거나 D열에 결과를 실시간 기록할 때 오류가 발생하는 경우입니다.

### 증상:
- `load_from_active_excel` 실행 시 엑셀 데이터를 가져오지 못함.
- `write_result_to_excel` 동작 시 프로그램이 멈추거나 에러 발생.

### 해결 방법:
- **엑셀 프로그램 확인**: 본 프로그램은 Microsoft Excel 정품 프로그램이 PC에 실행 중인 상태에서만 연동됩니다. (한컴오셀, 웹 브라우저용 엑셀, 리브레오피스 등은 연동되지 않습니다.)
- **엑셀 시트 편집 모드 해제**: 엑셀 시트 안에서 커서가 깜빡이며 입력 중인 상태(편집 모드)일 경우, 연동이 차단됩니다. `ESC` 키를 눌러 셀 선택 상태로 만든 뒤 다시 시도하세요.
- **권한 일치 (Windows)**: 엑셀 프로그램이 '관리자 권한'으로 실행되었으나 매크로는 일반 권한으로 실행된 경우 COM 연동이 차단됩니다. 둘 다 동일한 권한 수준으로 실행하세요.

## 3. 마우스 갑작스러운 이동 및 폭주 (Fail-Safe 작동)
매크로 실행 중에 마우스가 오작동하거나 멈추지 않는 긴급 상황에서의 강제 종료 기능입니다.

### 증상:
- 화면 구석으로 마우스가 튀면서 매크로가 강제로 중지됨.
- 로그에 `FailSafeException: PyAutoGUI fail-safe triggered` 에러 기록됨.

### 설명 및 조치:
- **안전장치(Fail-Safe)**: PyAutoGUI는 마우스 포인터를 **화면 모퉁이 네 곳 중 하나(좌측 상단 [0,0] 등)**로 빠르게 이동시키면 매크로를 즉시 강제 중지하도록 설계되어 있습니다.
- **예방**: 매크로가 도는 중에는 마우스를 억지로 붙잡고 움직이지 마세요. 긴급하게 멈추고 싶을 때만 마우스를 화면 구석(끝)으로 확 밀어주면 중지됩니다.

## 4. 디스코드 알림이 오지 않는 경우
사용 상태 모니터링 웹훅 카드 전송이 오지 않을 때입니다.

### 해결 방법:
1. `🔧 관리자 설정` 패널을 열어 디스코드 웹훅 주소가 정확하게 입력되었는지 확인합니다.
2. 입력창 우측의 **[🧪 테스트 전송]** 버튼을 클릭하여 결과 팝업 메시지를 확인합니다.
3. 팝업이 **"성공"**임에도 안 온다면 디스코드 채널의 알림 차단 여부나 권한을 확인하고, **"실패"**라면 웹훅 URL 오타 또는 네트워크 방화벽 차단 상태를 확인하십시오.

## 5. 진단 프로그램 활용
환경을 빠르게 점검하고 싶다면, 동봉된 `diagnose_env.py` 파일을 터미널에서 실행해 보세요.
```bash
python3 diagnose_env.py
```
필수 라이브러리 설치 현황 및 권한 안내를 한눈에 모니터링할 수 있습니다.

## 6. 발송 제한 가드 작동으로 일시정지되는 경우
### 증상:
- 발송이 진행되다가 `⚠️ 발송 제한 도달로 일시정지됨` 경고 팝업이 뜨고 멈춤.

### 설명 및 해결 방법:
- **요금제 보호**: 통신사 스팸 방지 정책(일일 발송 150건/500건 한도)에 도달해 소중한 휴대폰 회선이 스팸 번호로 지정 및 정지되는 불상사를 막기 위해 안전을 위해 설계된 기능입니다.
- **해결 방법**: 발송을 계속 이어나가고 싶다면, 화면 상단의 `⚠️ 발송 수량 제한` 체크박스를 해제하거나 한도 숫자를 더 높게 조정한 뒤, 하단의 **[▶ 재개]** 버튼을 클릭하여 다시 가동하십시오.

## 7. 엑셀 결과 보고서 저장 시 에러가 발생하는 경우
### 증상:
- `📊 결과 엑셀 보고서 저장` 실행 시 엑셀 파일 저장 실패 또는 에러 팝업 노출.

### 해결 방법:
- **파일 중복 잠금 해제**: 저장하려는 경로에 동일한 이름의 결과 보고서 엑셀 파일이 이미 Excel 프로그램 등으로 실행/열려 있는 경우, 덮어쓰기 쓰기 권한이 차단되어 에러가 납니다. 기존 엑셀 창을 모두 종료한 뒤 다시 시도해 주세요.
- **저장 위치 확인**: 바탕화면, 문서 폴더 등 쓰기/저장 권한이 완벽히 허용된 로컬 디렉토리 경로에 저장하십시오.

---

## 8. "🚫 접근 차단됨" 경고가 뜨며 실행이 불가능한 경우
### 증상:
- 프로그램 실행 시 `비인가 기기 또는 비정상적인 접근으로 인해 사용이 차단되었습니다.` 에러 팝업창이 출력되며 강제 종료됨.

### 원인 및 대처 방법:
- **블랙리스트 작동**: 관리자가 원격 라이선스 제어 파일에 본인의 기기(PC명/사용자명) 또는 공인 IP 주소를 차단 목록(`[BLACKLIST]`)에 추가한 상태입니다.
- **해결 방법**: 무단 유출 또는 공유 기기로 감지되어 차단되었을 가능성이 높으므로, 판매자/관리자에게 본인 PC의 식별자(예: `heebonui-MacBookPro.local/root`)를 전달하고 차단 해제를 요청하십시오.
